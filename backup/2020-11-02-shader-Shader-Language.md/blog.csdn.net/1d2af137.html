GLSL环境搭建

开始学习OpenGL编程指南（第八版)。这本书从一开始就开始引入着色器(shader)的相关知识，所以要使用GLSL(OpenGL Shading Language)。而书中并没有详细的讲编程环境的搭建过程。最后，经过一番折腾，终于把环境搭建好了。

搭建平台介绍

  • Windows 8.1 专业版 64位操作系统
  • Visual Studio 2015

下边就是具体步骤：

安装步骤

安装freeglut

由于glut已经不再更新，而且比较老旧，书中使用的是freeglut。

1.下载msvc版本的freeglut，下载地址here。也可以根据自己的需求进行下载。
2.解压到自定义的某个文件夹,eg:E:\ALearn\OpenGL\freeglut。
3.visual studio的根目录位：E:\Microsoft Visual Studio 14.0

freeglut.dll 放在C:\Windows\System32路径下（因为系统%PATH%路径是这里，不是C:\Windows\SysWOW64）
freeglut.lib 放在E:\Microsoft Visual Studio 14.0\VC\lib\amd64路径下
将freeglut\inlcude文件夹下的GL复制到E:\Microsoft Visual Studio 14.0\VC\include文件夹下

在建立新的程序时，solution platforms选择x64.这样程序才会运行成功。
不然会出现LNK1104:connot open file "freeglut_static_vs2010_d.lib"的链接错误。

安装glew

  1. 下载glew Binaries文件到本地，解压缩。
  2. 文件的放置同freeglut中文件的放置。
  3. 在project->properties->linker->input->Additional Dependencies中填入freeglut.lib;glew32.lib

不然会出现很多unresolved symbols链接错误。

测试程序

下载测试程序，选择x64平台运行程序。
