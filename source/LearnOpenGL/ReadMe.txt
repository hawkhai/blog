========================================================================
WIN32 应用程序：LearnGL 项目概述
========================================================================

应用程序向导已为您创建了此 LearnGL 应用程序。  

本文件概要介绍组成 LearnGL 应用程序的每个文件的内容。


LearnGL.vcproj
这是使用应用程序向导生成的 VC++ 项目的主项目文件。 
它包含生成该文件的 Visual C++ 的版本信息，以及有关使用应用程序向导选择的平台、配置和项目功能的信息。

LearnGL.cpp
这是主应用程序源文件。

/////////////////////////////////////////////////////////////////////////////
应用程序向导创建了下列资源：

LearnGL.rc
这是程序使用的所有 Microsoft Windows 资源的列表。它包括 RES 子目录中存储的图标、位图和光标。此文件可以直接在 Microsoft Visual C++ 中进行编辑。

Resource.h
这是标准头文件，它定义新的资源 ID。
Microsoft Visual C++ 读取并更新此文件。

LearnGL.ico
这是用作应用程序图标 (32x32) 的图标文件。
此图标包括在主资源文件 LearnGL.rc 中。

small.ico
这是一个图标文件，其中包含应用程序图标的较小版本 (16x16)。此图标包括在主资源文件 LearnGL.rc 中。

/////////////////////////////////////////////////////////////////////////////
其他标准文件：

StdAfx.h, StdAfx.cpp
这些文件用于生成名为 LearnGL.pch 的预编译头 (PCH) 文件和名为 StdAfx.obj 的预编译类型文件。

其他注释：

应用程序向导使用“TODO:”指示应添加或自定义的源代码部分。
