Thus, there is a need for pre- processing of the skin images to remove these obstructing hair. This sample project aims to remove the hair noise from the skin image with the help of Morphological filtering. ## Steps involved in the DHR algorithm - Convert the color image to a grayscale version. - Applying Morphological Black-Hat transformation on the grayscale image - Creating the mask for InPainting task - Applying inpainting algorithm on the original image using the mask prepared from the grayscale image in step 3 ## Examples: ### Example 1 Original Image | [![alt
