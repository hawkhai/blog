线性代数之——行图像和列图像

1. 线性方程组的几何解释

线性代数的中心问题就是解决一个方程组，这些方程都是线性的，也就是未知数都是乘以一个数字的。

3x − x + 22y = 1y = 11x − 2y = 13x + 2y = 11

针对上面的方程组，如果我们一行一行来看的话，那么第一个方程 x−2y=1x−2y=1 表示二维平面的一条直线。点 (1, 0) 是方程的一个解，点 (3, 1) 也是方程的一个解，因此它们都位于这条直线上。

同理，第二个方程 3x+2y=113x+2y=11 也表示二维平面的一条直线，这两条直线的交点也就是上述方程组的解，因为它同时满足了方程一和方程二 。

另外，我们也可以将上述方程写成向量的形式：

x[13]+y[−22]=[111]=bx[13]+y[−22]=[111]=b

问题就变成了寻找左边两个向量的一个特定线性组合来产生右边的向量。

若将上述方程组表示成矩阵的形式，就是：

Ax=b↔[13−22][xy]=[111]Ax=b↔[1−232][xy]=[111]

行图像就是对矩阵 AA 的行进行处理，而列图像则是矩阵 AA 的列的线性组合。

2. 三个未知数三个方程

针对三个未知数 x,y,zx,y,z，我们有三个线性方程：

26x + x + x − 253y + y + y + 32z = 6z = 4z = 2x + 2y + 3z = 62x + 5y + 2z = 46x − 3y + z = 2

在行图像中，每个方程产生一个三维空间中的平面。第一个平面和第二个平面相交于一条直线 LL，然后第三个平面和这条直线又相交于一点，也就是方程组的解。

在列图像中，我们要寻找左边向量的一个线性组合。在这里，我们可以非常容易地看到方程组的解，(6, 4, 2) 为 (3, 2, 1) 的 2 倍，因此解就为 (0, 0, 2)。

x⎡⎣⎢126⎤⎦⎥+y⎡⎣⎢25−3⎤⎦⎥+z⎡⎣⎢321⎤⎦⎥=⎡⎣⎢642⎤⎦⎥x[126]+y[25−3]+z[321]=[642]

表示成矩阵的形式，就是

Ax=b↔⎡⎣⎢12625−3321⎤⎦⎥⎡⎣⎢xyz⎤⎦⎥=⎡⎣⎢642⎤⎦⎥Ax=b↔[1232526−31][xyz]=[642]

AA 乘以 xx 可以看成是 xx 和矩阵的行的点积

Ax=⎡⎣⎢(row1)⋅x(row2)⋅x(row3)⋅x⎤⎦⎥Ax=[(row1)⋅x(row2)⋅x(row3)⋅x]

也可以看成是矩阵的列的线性组合

Ax=x(column1)+y(column2)+z(column3)Ax=x(column1)+y(column2)+z(column3)
